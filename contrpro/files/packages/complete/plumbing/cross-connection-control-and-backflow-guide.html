Section 1 — Why Backflow Matters: Hazard Mechanics

Backflow occurs whenever water flows in the reverse of intended direction. Two physical mechanisms drive it:

  • Backpressure — downstream pressure (boiler, pump, elevated tank, chemical feed) exceeds upstream supply pressure. The high side pushes back into the potable supply.
  • Backsiphonage — upstream supply pressure drops below atmospheric (main break, fire flow, repair drawdown). Downstream non-potable water siphons into the main.

Both mechanisms are documented in fatality + illness incidents. The EPA + CDC track several hundred reportable cross-connection incidents per year nationally. The mitigating device must match the hazard mechanism AND the hazard severity.

WARNING. Failure to install the correct backflow device, or installing it backwards, has caused documented illness and death. The plumbing contractor is responsible for selection, installation per manufacturer's instructions, AND arranging initial testing by a licensed cross-connection control specialist BEFORE the system goes into service.
Section 2 — Hazard Classifications

Most jurisdictions use a two-tier (or three-tier) hazard system. Match the highest plausible severity for the application.

ClassDefinitionExamples
High (Health) Backflow would cause illness, death, or serious bodily harm. Sewage / wastewater, boilers with chemicals, cooling towers with biocides, lab reagents, irrigation with chemical injection, medical-gas systems, fire systems with antifreeze.
Low (Non-Health / Pollution) Backflow would be objectionable but non-toxic. Food + beverage processing, soda fountain water (without injection), non-chemical irrigation, non-chemical fire systems, secondary heat exchangers.
Air-gap acceptable Physical separation of supply outlet from receiving vessel. Indirect waste from washer / icemaker, RPZ relief discharge, mop sink fill.

Section 5 — Installation Requirements
5.1 RPZ + DCV — physical orientation
  • Install horizontally with the body level. Most manufacturers DO NOT permit vertical installation of RPZ — read the instructions and reject any field directive otherwise.
  • Minimum 12-inch clearance below the relief port (some manufacturers require more). The relief port discharges visibly and audibly during normal operation.
  • Minimum 6-inch clearance around the body for testing access.
  • Test cocks (4 on RPZ, 4 on DCV) accessible without removing the device.
  • Strainer upstream — recommended on DCV; required on RPZ for many manufacturers.
  • Isolation valves upstream + downstream (ball or gate, full-port).
  • NEVER plumb the relief port back to the sanitary line directly — must discharge to an indirect waste with an air gap of 2× pipe dia (1" min).
  • Adequate floor drain to accept the relief discharge volume.
5.2 Cold-climate freeze protection
  • Install indoors where possible.
  • Outdoor enclosure rated for the device + heated to 40°F minimum.
  • Heat tape with thermostat on body + test cocks; insulation on body but not over relief port discharge path.
  • Drain valves at low points; isolation valves accessible from inside enclosure.
5.3 PVB / SVB installation
  • Install 6 inches minimum above the highest downstream outlet (12-inch is best practice — confirm AHJ + manufacturer).
  • No isolation valve downstream of PVB (would create the same backpressure scenario the PVB is NOT rated for).
  • Outdoors only on dedicated irrigation use, not building potable.
Section 6 — Initial Field Test Procedures

Tests must be performed by a state-licensed cross-connection control specialist using a calibrated 3-needle differential gauge. The procedures below are the ASSE-standard test sequence. Document in the Backflow Test Log XLSX (Initial Test tab).

6.1 RPZ test sequence (ASSE 1013 abbreviated)
  1. Install gauge hoses on test cocks. Open all four cocks one at a time to clear lines.
  2. Test 1 — Check Valve #1. Close downstream isolation. Open test cock #4 to drop downstream pressure. Read static differential at test cock #2 / #3. PASS criteria: ≥1.0 psid (some specs require ≥2.0 psid).
  3. Test 2 — Relief Valve Opening. Slowly close upstream isolation valve while reading the differential gauge. Relief should open at ≥2.0 psid AND before the gauge reads zero. PASS criteria: opens at ≥2.0 psid.
  4. Test 3 — Check Valve #2. Equalize pressures + close test cock #4. Verify second check holds tight against zero differential.
  5. Record: device location, serial number, model, all three test values, pass/fail, date, tester name + license #.
  6. If any test fails: tag device "OUT OF SERVICE — DO NOT USE" + arrange repair.
6.2 DCV test sequence (ASSE 1015 abbreviated)
  1. Open test cock #4 to drop downstream pressure.
  2. Test 1 — Check #1. Verify ≥1.0 psid differential held.
  3. Test 2 — Check #2. Close test cock #4, open #3 + verify ≥1.0 psid held by check #2.
  4. Record results.
Practice note — gauge calibration. Test gauges must be calibrated within the previous 12 months. Calibration certificate must accompany every test report. AHJs increasingly check for current calibration — a test performed with an out-of-cal gauge will be rejected.
Section 7 — Annual Testing & Reporting

Most jurisdictions require annual testing of every continuous-pressure backflow device in commercial service. The owner is typically responsible for arranging the test; the plumbing contractor may perform under separate service contract.

  • Test interval — 12 months from initial test, then annually. Some jurisdictions require 6-month testing for hospitals + high-risk locations.
  • Testing party — state-licensed cross-connection control specialist with current gauge calibration.
  • Report destinations — typically: (1) building owner / facility manager, (2) water purveyor, (3) local AHJ. Confirm with each jurisdiction at the initial install.
  • Report retention — 5 years minimum. Many AHJs require longer.
  • Out-of-service tagging — failed devices must be tagged AND the owner notified in writing within 24 hours. Repair or replace within 30 days.

Section 9 — Documentation & Submission

Use the companion Backflow Test Log XLSX to capture every test. Each test record should include:

  • Device identifier — owner-assigned ID + serial number.
  • Location — building, floor, room.
  • Make + model + size.
  • Hazard application (water service, boiler, irrigation, etc.).
  • Test date + tester name + license #.
  • All three test readings (check 1 differential, relief opening, check 2 held).
  • Pass / fail.
  • Gauge serial # + calibration date.
  • Repairs performed if applicable.
  • Owner notification + signature.
  • Submission confirmation to water purveyor + AHJ.
Section 10 — Common Field Errors
  1. Installing an RPZ vertically (most are horizontal-only).
  2. Plumbing the relief port directly to drain (must be indirect with air gap).
  3. Selecting DCV where RPZ is required (health-hazard application).
  4. Using AVB on continuous-pressure application (12-hour-max rating).
  5. No clearance below relief port (some require 18" minimum).
  6. Test cocks blocked by adjacent piping (no testing access).
  7. Reversed flow direction (read the arrow on the body).
  8. Hose-bibb vacuum breaker missing on outdoor sillcocks.
  9. Service-entry RPZ installed without bypass — every test interrupts the building's water.
  10. Skipping initial test because "we tested at the bench" (not equivalent — AHJ wants the field test).
