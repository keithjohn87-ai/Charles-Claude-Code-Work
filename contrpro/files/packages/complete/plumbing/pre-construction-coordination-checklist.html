USE FOR EVERY NEW PROJECT — BEFORE FIRST PRODUCTIVE DAY ON SITE

Print this checklist for the pre-construction coordination meeting. Walk every line item with the GC superintendent, the EOR's plumbing engineer, and the AHJ plumbing inspector where available. Anything unresolved becomes an action item with an owner and a date. Sign at Section 14. File the signed copy with the project's contract documents.

Purpose. Catch every coordination gap BEFORE materials are ordered and crews mobilize. Section 2 — Contract Documents & Scope Confirmation
  1. Signed plumbing subcontract on file (or owner-direct PO).
  2. All exhibits attached (drawings, specs, schedule, insurance reqs, scope statement, exclusions).
  3. Scope boundaries with adjacent trades documented in writing (HVAC, FP, EC, site utility).
  4. Inclusions / exclusions list signed by both parties.
  5. Change-order process documented — who can authorize? What's the dollar threshold? Lead time required?
  6. Pay-application schedule — submission day, retention %, lien-waiver type (conditional/unconditional).
  7. Liquidated damages — daily rate? Curing process?
  8. Warranty period documented — start date trigger, scope of warranty work.

Section 4 — Drawing & Spec Review (Plumbing + Coordination)
Plumbing drawings
  • Current revision set on hand at GC trailer + plumbing field trailer.
  • All RFIs to date logged + dispositioned.
  • P-0.0 / 1.0 series plumbing plans reviewed against P-2.0 / 3.0 details.
  • Fixture schedule + cut sheets cross-referenced against specs (matched manufacturers).
  • Riser diagrams checked for vertical clearance through structure.
  • Cross-connection control table identifies device + hazard tier for each fixture / system.
  • Sleeve + opening schedule reconciled with structural / architectural.
Coordination — overlay against M, E, FP, S
  • Above-ceiling overlay (BIM or marked-up 2D) reviewed — clashes flagged.
  • Below-slab utility overlay reviewed — site utility congestion, depth conflicts.
  • Wall penetrations through rated assemblies — UL listing identified per location.
  • Roof penetrations — coordinated with roofing manufacturer mock-up.
  • Trapeze hanger locations — structural attachment approved by EOR.

Section 10 — Safety Program Confirmation
  • Site-Specific Plumbing Plan (SSPP) reviewed + signed.
  • Confined-space program — entry permit form, atmospheric meter calibration, rescue services identified.
  • Hot-work permit program — daily permit, fire watch, 30-min cool-down.
  • Excavation Competent Person identified + Subpart P training current.
  • Hazcom / GHS — SDS available for chemicals (flux, primer, cement, chlorine).
  • Lockout / tagout program — written procedures, locks issued, training current.
  • PPE issued — eye, gloves (cut + chemical), hard hats, hi-viz, hearing where required.
  • Toolbox-talk schedule — weekly minimum, documented attendance.
  • First-aid kit location + AED location confirmed.
  • Heat / cold stress protocol per local climate.
