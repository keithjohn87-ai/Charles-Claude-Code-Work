ContrPro Electrical Trade Pack — Complete Tier
====================================================

Trade-specific delta on top of the Universal Sub Suite for commercial
electrical contractors. Full division scope (raceway, conductors, gear,
lighting, controls, low-voltage cross-references) — field installation
only, no fabrication-shop pre-assembly.

USE THIS WITH THE UNIVERSAL SUB SUITE.

====================================================
What you get — 6 deliverables, 10 files
====================================================

1. Electrical Bid Estimator
   (Electrical_Bid_Estimator.xlsx)
   10-tab NECA-labor-unit-based bid estimator. CSI Division 26 primary
   with cross-references to 27 (communications) and 28 (electronic safety
   + security). 90+ Material Takeoff line items; 50+ Labor activities.

2. Site-Specific Electrical Safety Plan
   (site-specific-electrical-plan.html / .docx)
   16-section project-specific roadmap — NEC compliance, NFPA 70E electrical
   safety, LOTO, arc-flash boundaries, energized work, AHJ inspection
   sequencing, multi-trade coordination, acceptance testing.

3. Pre-Construction Coordination Checklist
   (pre-construction-coordination-checklist.html / .docx)
   14-section pre-mobilization meeting agenda + minutes. Attendance roster,
   contract review, permit + AHJ + utility coordination, drawing + spec
   review, long-lead material tracking (gear / gen / ATS), submittal
   schedule, site logistics, multi-trade coordination, inspection sequence,
   safety program, insurance/bonds/licenses, manpower forecast, action items.

4. LOTO + Arc-Flash + Energized Work Guide + Log
   (loto-arc-flash-and-energized-work-guide.html / .docx +
    EWP_and_LOTO_Log.xlsx)
   10-section compliance guide + 6-tab log workbook.
     HTML: hierarchy of controls, LOTO step-by-step, voltage verification
     (test-the-tester), EWP issuance, arc-flash boundary + PPE categories
     1-4, shock protection boundaries, arc-rated PPE matrix, qualified vs
     unqualified person, NEC 110.16 labeling, incident response.
     XLSX: LOTO Log (60 rows), EWP Log (40 rows), Voltage-Test Log,
     PPE Inventory + Calibration Tracking, Reporting Summary.

5. Conduit & Cable Installation Guide + Pull/Test Log
   (conduit-and-cable-installation-guide.html / .docx +
    Pull_and_Insulation_Test_Log.xlsx)
   10-section field reference + 5-tab test-log workbook.
     HTML: raceway selection by environment, conduit fill (Chapter 9 Table 1
     +derating), support spacing per type, bend radius + pull boxes,
     conductor selection (ampacity + voltage drop), pull tension limits,
     splices + terminations, voltage-class separation, underground raceway
     (300.5), megger insulation testing.
     XLSX: Cable Pull Log (60 rows), Insulation Test Log (60 rows),
     Acceptance Sign-Off, Reporting Summary.

6. Grounding & Bonding Guide + Ground Resistance Test Log
   (grounding-and-bonding-guide.html / .docx +
    Ground_Resistance_Test_Log.xlsx)
   10-section NEC Article 250 reference + 6-tab test-log workbook.
     HTML: grounding vs bonding, Grounding Electrode System (250.50-66),
     EGC requirements, bonding (service / building / metal piping),
     sizing tables (Table 250.66 + 250.122), 3-point fall-of-potential
     ground resistance testing (IEEE 81), connection methods, special
     locations (healthcare / pool / solar / generator / hazardous), common
     errors, test + acceptance procedures.
     XLSX: Electrode Inventory, 3-Point Test Log (40 rows), Clamp-On Test
     Log, Soil Resistivity (Wenner 4-point), Reporting Summary.

====================================================
Field-only scope
====================================================

Calibrated for field installation. Switchgear assembly + factory acceptance
testing, specialty fabricated bus assemblies, and modular substation work
are typically subbed and entered on the Subcontract tab of the estimator.

Full scope WITHIN field installation includes: service entrance + meter,
switchgear + panelboards (set + connect), conductors (pull + terminate),
raceway (interior + underground), grounding electrode system, lighting +
lighting controls, devices, motor controls + VFDs, standby generator +
ATS (typically install + start-up by vendor; coordinate scope split),
fire alarm if NICET-certified in scope, low-voltage cross-trade items.

====================================================
Important reminders
====================================================

References NEC 2023 / NFPA 70E 2024 / OSHA 29 CFR 1926 Subpart K / 29 CFR
1910.147 LOTO / NEC 110.16 labeling / IEEE 81 + 142 grounding / NETA
acceptance testing without reproducing copyrighted material. NOT legal
advice and NOT a substitute for the Master Electrician of Record's
professional judgment, the AHJ's project-specific direction, an engineered
arc-flash study by a qualified PE, or NETA-certified acceptance testing
for critical installations.

Before first use on any new project, have construction counsel + your
master electrician review the tailored documents. Code references current
as of January 2026 — verify against the AHJ-adopted edition + any local
amendments.
