Section 2 — Contract Documents
  1. Signed subcontract on file with all exhibits.
  2. Scope boundaries with adjacent trades documented (FP / Plumb / Mech).
  3. Inclusions / exclusions list signed.
  4. Change-order process documented (authorization + dollar threshold + lead time).
  5. Pay-app schedule + retention % + lien-waiver type.
  6. Liquidated damages + curing process documented.
  7. Warranty period documented.

Practice note — NEC edition delta. Drawings stamped under prior NEC + constructed under newer NEC creates inspection failures. Document deltas at pre-con. Common 2020→2023: GFCI scope (210.8), AFCI scope (210.12), commercial surge (230.67), EVSE (625), energy storage (706).
Section 4 — Drawing + Spec Review
Electrical drawings
  • Current revision set at GC + EC trailers.
  • All RFIs dispositioned.
  • Single-line diagram cross-checked against panel schedules + load summary.
  • Fault-current calc + breaker AIC ratings verified.
  • Arc-flash study reviewed (if engineered study provided).
  • Voltage drop calc verified against branch + feeder lengths.
  • Conduit + cable schedules cross-checked.
  • Lighting layout vs. reflected ceiling plan reconciled.
  • Sleeve + opening schedule reconciled with structural + arch.
Coordination overlay — M / P / FP / S
  • Above-ceiling overlay reviewed; clashes flagged.
  • Below-slab utility overlay reviewed.
  • Wall + floor penetrations through rated assemblies — UL listings identified.
  • Trapeze + structural-attachment locations approved by EOR.
  • Working clearances (NEC 110.26) verified for all gear locations.

Section 10 — Safety Program Confirmation
  • SSEP reviewed + signed.
  • LOTO program — locks/tags issued, training current.
  • NFPA 70E training current; EWP forms on site; arc-rated PPE inventoried.
  • Arc-flash study reviewed; labels ready for installation.
  • Hot work permit program.
  • Excavation Competent Person identified.
  • Lift / scissor-lift training current.
  • GFCI on every temp-power circuit verified.
  • Hazcom / SDS for adhesives, sealants, cleaners.
  • Toolbox-talk schedule; first-aid + AED locations.
