TAB + COMMISSIONING GUIDE

Section 1 — TAB vs Cx
TABCommissioning (Cx)
Purpose Measure + adjust airflows, water flows, and pressures to match design Verify systems function per Owner's Project Requirements (OPR) + Basis of Design (BOD)
Performer NEBB or AABC certified firm (typically a sub) Cx Authority (CxA) — owner-direct, independent of mech contractor
Reference Standard NEBB Procedural Standard / AABC National Standard ASHRAE Guideline 0 (overall Cx process), ASHRAE Guideline 1.1 (HVAC Cx)
Timing Pre-substantial; after install + before final inspection Spans design through 12-month post-occupancy
Output TAB report with as-measured values Cx report — FPT results, deficiency log, training records
Typical sequence
  1. Mechanical install complete + pressure tested + filters installed.
  2. Pre-functional checklists (PFC) — installer verifies + signs each piece of equipment.
  3. Equipment start-up — manufacturer's tech where required.
  4. Air-side TAB — measure + adjust supply / return / exhaust flows.
  5. Water-side TAB — measure + adjust hydronic flows.
  6. Sequence of Operations (SoO) review with CxA + EOR + controls.
  7. Functional Performance Tests (FPT) — system + integrated.
  8. Deficiency correction + re-test.
  9. Owner training + O&M handover.
  10. 12-month seasonal verification (cooling + heating).
Section 2 — TAB Methodology (NEBB / AABC)
  • NEBB-certified or AABC-certified firm (verify cert + project-specific TAB technician cert).
  • Instrument calibration current — required on every project. Verify cal sticker.
  • Pre-TAB inspection: filters installed (clean), dampers open, system per design.
  • TAB report template per NEBB / AABC standard.
  • Project-specific tolerance — typically ±10% of design value for airflow, ±5% for water flow (verify spec).
Section 3 — Air-Side TAB Procedures
  1. Start at the AHU/RTU: total supply + return + OA airflows. Adjust at unit volume control or VFD to reach design.
  2. Trunk + main branches — Pitot traverse where access permits. Measure velocities → calculate cfm.
  3. Terminal units (VAV / FCU) — measure inlet + adjust min/max settings.
  4. Diffusers + grilles — hood capture method or Alnor traversing. Adjust dampers to distribute design cfm.
  5. Re-balance trunk + main after diffusers — minor iteration.
  6. Outdoor air (OA) — verify per fixed minimum or DCV setpoint.
  7. Exhaust flows — kitchen, lab, bathroom, parking ramp.
  8. Pressure relationships — building, room, lab — measured + recorded.
  9. Document each terminal + each piece of equipment on TAB Air-Side Log.
Section 4 — Water-Side TAB Procedures
  1. Pump GPM + head — measure at design + verify against pump curve.
  2. Riser flows — adjust via balancing valves to design.
  3. Coil flows — measure via P-T plugs + pressure-drop or via balancing-valve gauge.
  4. Boiler + chiller — verify delta-T per design.
  5. Cooling tower — verify approach + range.
  6. Document on TAB Water-Side Log.
Section 5 — Cx Authority Process per ASHRAE Guideline 0
  • OPR — Owner's Project Requirements document (often by CxA).
  • BOD — Basis of Design (by EOR).
  • Design Review — CxA reviews design against OPR.
  • Cx Plan — issued by CxA at construction start. Includes Cx schedule, equipment list, FPT scope, training plan.
  • Installation Verification — Pre-Functional Checklists (PFC) by installer + verified by CxA.
  • FPT — Functional Performance Tests per equipment + system.
  • Integrated Systems Test — multi-system test (life-safety + smoke + HVAC integration).
  • Training + Documentation — owner training; final Cx report; warranty period follow-up.
  • Seasonal Verification — opposite-season testing (typically 9-12 months post).
Section 6 — Sequence of Operations Review
  • Review SoO from BAS submittal with CxA + EOR + controls before FPT.
  • Each operating mode (occupied / unoccupied / warm-up / setback / morning-warm-up / smoke mode / fire-alarm) — define triggers + expected response + override behavior.
  • Setpoints + dead-bands documented.
  • Alarm thresholds + escalation defined.
  • BAS graphics + point lists reviewed for completeness.
  • Owner-facing dashboards reviewed.
Section 7 — Functional Performance Tests (FPT)
  • Equipment-specific FPT — each AHU, RTU, chiller, boiler, pump, fan, VFD per spec template.
  • Stepwise tests: start, run at min/normal/max output, stop, alarms, BAS reporting.
  • Verify safeties — low-temp cutout, smoke detector trip, freeze stat, high-static cut-out.
  • Verify staging on multi-piece equipment (lead-lag, alternation).
  • Verify control-loop response (no hunting, stable PID).
  • Document on FPT Log; deficiency → re-test after correction.
Section 8 — Integrated Systems Test
  • Integration of multiple systems: HVAC + fire alarm + smoke control + life-safety.
  • Test smoke-mode response: smoke detector trips → HVAC shutdown + dampers operate + stairwell pressurization.
  • Test fire-alarm interface: dampers close, emergency exhaust starts, AHUs shut down per SoO.
  • Test power-loss + emergency restoration: equipment restart sequence per SoO.
  • Witness by CxA + AHJ where required by life-safety code.
  • Document on Integrated Test Log.
Section 9 — Deficiency Log + Re-Test
  • Log every deficiency identified during FPT / IST.
  • Assign owner + due date.
  • Re-test after correction; don't simply mark "done."
  • Open deficiencies → withhold from substantial completion sign-off.
  • Categorize by severity: punch (cosmetic), functional (impacts operation), critical (impacts safety / OPR compliance).
Section 10 — Final Reports + Owner Training
  • TAB Final Report — by NEBB / AABC firm. Includes as-measured values for every terminal + equipment + system pressure.
  • Cx Final Report — by CxA. Includes OPR + BOD, FPT results, deficiency log closeout, training records, recommendations.
  • Owner Training — typically 8+ hours covering system overview, sequence of operations, BAS interface, alarms, routine maintenance. Documented with sign-in sheet.
  • O&M Manual integration — TAB + Cx reports become part of O&M manual.
  • Warranty period — Cx-tracked deficiencies addressed under warranty; 12-month seasonal verification typical.
