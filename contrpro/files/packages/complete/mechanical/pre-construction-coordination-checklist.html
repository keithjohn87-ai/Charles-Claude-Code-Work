Practice note — energy-code envelope. ASHRAE 90.1 / IECC requires equipment efficiency, controls (DCV, economizer, occupancy), and pipe + duct insulation minimums. Verify the spec'd equipment meets the energy path the design used (prescriptive vs performance). Field substitutions can break the compliance basis.
Section 4 — Drawing + Spec Review
  • Current revision set at GC + mech trailers.
  • All RFIs dispositioned.
  • Load calcs + equipment selection confirmed against spec.
  • Above-ceiling overlay reviewed (M / E / P / FP).
  • Roof-curb + structural-support coordination.
  • Equipment service access + working clearances verified.
  • Ventilation rates per ASHRAE 62.1 confirmed.
  • Sequence of operations reviewed with controls + EOR.
  • SOV per CSI section established for billing.

Section 7 — Site Logistics + Rigging Plan
  • Site access + parking + lay-down area for ductwork + equipment.
  • Crane lifts coordinated (date / capacity / ground-bearing).
  • Roof access + fall protection plan.
  • Critical-lift criteria identified for any lift > 75% capacity.
  • Sheet-metal staging + handling.
  • Temp power + lighting for mech rooms.
  • Compressed-air or nitrogen supply for testing.
  • Refrigerant cylinder storage area (ASHRAE 15-compliant).

Section 10 — Safety Program Confirmation
  • SSMP reviewed + signed.
  • EPA 608 certs current; refrigerant SOP on site.
  • Hot work permit program + nitrogen-purge SOP for brazing.
  • Confined-space program for mech rooms / plenums.
  • Crane / rigging plan reviewed + operators certified.
  • Fall protection for roof work.
  • LOTO program + locks/tags issued.
  • Hazcom / SDS for refrigerants, fluxes, glycol, chemicals.
  • Toolbox-talk schedule; first-aid + AED locations.
