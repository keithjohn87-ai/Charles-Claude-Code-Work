ContrPro Mechanical Trade Pack — Complete Tier
====================================================

Trade-specific delta on top of the Universal Sub Suite for commercial
mechanical (HVAC + hydronic + refrigeration) contractors. Full division
scope (ductwork, hydronic, gas, refrigerant, equipment, controls) — field
installation only, no shop-prefab.

USE THIS WITH THE UNIVERSAL SUB SUITE.

====================================================
What you get — 6 deliverables, 11 files
====================================================

1. Mechanical Bid Estimator
   (Mechanical_Bid_Estimator.xlsx)
   10-tab estimator with per-CFM + per-ton + per-LF + per-SF anchors.
   CSI Division 23 primary with cross-references to 22 (plumbing),
   26 (electrical), and 27 (BAS / controls). 100+ Material Takeoff
   line items; 60+ Labor activities.

2. Site-Specific Mechanical Safety + QC Plan
   (site-specific-mechanical-plan.html / .docx)

   heavy equipment lifting + rigging, working at heights, hazmat,
   IAQ + construction-phase IAQ management (ASHRAE 62.1), multi-trade

   touchpoints, emergency response, stop-work triggers.

   contract review, permit + AHJ + energy code, drawing + spec review,
   long-lead equipment tracking, submittal schedule, site logistics +
   rigging plan, multi-trade coordination, inspection sequence, safety
   program, insurance + bonds + licenses + EPA 608 certs, manpower forecast,
   action items.

4. Refrigerant Handling + Hot Work Guide + Log
   (refrigerant-and-hot-work-guide.html / .docx +
    Refrigerant_and_HotWork_Log.xlsx)
   10-section compliance guide + 6-tab log workbook.
     HTML: EPA 608 certification levels, ASHRAE 34 refrigerant
     classifications (A1 / A2L / B1 etc.), recovery / recycling / reclaim,

     XLSX: Tech Certs, Recovery+Charge Log (60 rows), Leak+Repair Log,
     N2 Pressure+Vacuum Test, Hot Work Permits.

5. Ductwork + Piping Installation + Test Guide + Log
   (ductwork-piping-install-and-test-guide.html / .docx +
    Mechanical_Pressure_and_Leakage_Test_Log.xlsx)
   10-section installation reference + 6-tab test log.
     HTML: SMACNA duct gauge + reinforcement + support + sealing + leakage
     class, duct leakage testing (SMACNA DCS), hydronic piping material +
     hangers + pressure test + flush + fill, gas piping (IFGC 406), steam +
     condensate, acceptance documentation.
     XLSX: Duct Leakage Log, Hydronic Pressure Log, Gas Pressure Log,
     Steam Pressure Log, Flush + Fill Log.

6. TAB + Commissioning Guide + FPT Log
   (tab-and-commissioning-guide.html / .docx +
    TAB_and_Cx_FPT_Log.xlsx)

====================================================
Field-only scope
====================================================

Calibrated for field installation. Fab-shop sheet-metal production and
pre-fabricated piping rack assemblies are typically in-house or subbed;
material costs reflect delivered fabricated ductwork. Switchgear assembly
is electrical contractor's scope (cross-reference).

Full scope WITHIN field installation includes: ductwork (galvanized +
spiral + accessories), hydronic piping + pumps + tanks + valves,
gas piping (interior), refrigerant piping + charging (EPA 608), boilers +
heat exchangers, chillers + cooling towers, air-handlers + RTUs + ERVs,
terminal units (VAV / FCU / radiant), DX + VRF + mini-split, insulation
(duct + pipe), basic BAS sensors (controls programming typically subbed).
